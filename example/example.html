우편번호 검색

1. 도로명주소로 검색 : 도로명과 건물번호를 입력하세요. 예) 중앙로12길 345-6
2. 지번 주소로 검색 : "동" 또는 "리" 이름과 번지수를 입력하세요. 예) 우편동 123-45, 번호리 67
3. 건물명으로 검색 : 빌딩 또는 아파트 이름을 입력하세요. 예) 국제빌딩, 래미안, 주공3차아파트
4. 사서함으로 검색 : 사서함명과 번호를 입력하세요. 예) 광화문우체국사서함 123-4
    ※ 비슷한 지명이 전국에 여러 군데가 있는 경우 "삼척시 교동 60", "방배동 래미안", 등
    　 시/군/구/읍/면/동/리 이름을 넣어서 검색하시면 더욱 정확한 결과를 얻을 수 있습니다.
    ※ 건물명보다는 정확한 도로명주소 또는 지번 주소로 검색하시는 것이 확실합니다.
